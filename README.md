# EV&GAS - Android застосунок для порівняння витрат

Застосунок для порівняння фактичних витрат на електроавтомобіль та автомобіль на газу.

## 📋 Функціонал

- **Головний екран "Порівняння"** - візуальне порівняння витрат з кольоровою індикацією
- **Екрани "Електро" та "ГАЗ"** - контрольні показники для кожного типу авто
- **Архіви** - зберігання та редагування записів про витрати
- **Автоматичні розрахунки** - вартість 1 км, 100 км, загальний пробіг
- **Офлайн робота** - всі дані зберігаються локально

## 🛠 Технічний стек

- **Kotlin** - мова програмування
- **Jetpack Compose** - сучасний UI toolkit
- **Room Database** - локальне зберігання даних
- **MVVM Architecture** - архітектурний патерн
- **Coroutines & Flow** - асинхронність та реактивність
- **Material Design 3** - дизайн інтерфейсу

## 🚀 Встановлення та запуск

### Вимоги
- Android Studio Hedgehog або новіша
- JDK 17
- Android SDK 35
- Пристрій/емулятор з Android 15 (API 35)

### Кроки встановлення

1. Клонуйте репозиторій
2. Відкрийте проєкт в Android Studio
3. Синхронізуйте Gradle
4. Запустіть застосунок

## 📱 Використання

### Додавання запису
1. Перейдіть в "Електро" або "ГАЗ"
2. Натисніть "Архів"
3. Натисніть кнопку "+" (Новий запис)
4. Заповніть дату, пробіг та вартість
5. Натисніть "Зберегти"
